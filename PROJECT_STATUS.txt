│                      QUICK START                                 │
└──────────────────────────────────────────────────────────────────┘

BACKEND (2 minutes)
  $ cd backend
  $ docker-compose up -d postgres
  $ migrate -path migrations -database "postgresql://..." up
  $ go run cmd/server/main.go
  
  ✓ Server running at http://localhost:8080

FRONTEND (2 minutes)
  $ cd frontend
  $ cp .env.example .env
  $ flutter pub get
  $ flutter run
  
  ✓ App running on device/emulator
